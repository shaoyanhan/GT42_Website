Mosaic (Orthologous);

Annotation

Data details:

Function and structure annotation of transcripts.

Transcript (Isoform);

Download link: https://cbi.gxu.edu.cn/download/yhshao /GT42_web/annotation/filename

SNP

Data details:

SNP variations with sequencing evidence.

snp_final.txt: Confident SNPs selected based on both Iso-seq and RNA-seq evidence;

snp_isoseq.txt: SNPs with Iso-seq evidence counts;

snp_rnaseq.txt: SNPs with RNA-seq evidence counts;

snp_haplotype.txt: SNP type on specific haplotype;

Download link: https://cbi.gxu.edu.cn/download/yhshao /GT42_web/snp/filename

Transcript (Isoform);

Download link: https://cbi.gxu.edu.cn/download/yhshao /GT42_web/expression/filename
